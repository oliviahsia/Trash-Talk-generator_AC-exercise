# 幹話產生器

人總是莫名想要講點幹話，但當詞窮時這個小設計總是能帶給你無比方便。
今天想要對誰說點幹話呢？來試試吧！

## 功能

- 有三種不同職業直提的幹話
- 選取對應的照片選項可以產生隨機幹話

## 安裝方式

- 透過終端機輸入指令來要求下載專案

  ```js
  git clone https://github.com/oliviahsia/Trash-Talk-generator_AC-exercise.git
  ```

- 載入完成後進入資料夾開啟

  ```js
  cd trash_talk_generator
  ```

- 安裝套件

  ```js
  npm install
  ```

- 終端機中開啟伺服器，網址輸入 localhost:3000/ 即可開啟
